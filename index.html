回去的路上，我精神恍惚，半梦半醒间，以为自己在晋王府，梦里有人念道：鹤栖梧桐树，枝头变凤凰……或许只过了一瞬，或许过了一辈子那么久，忽然有什么漫上来，浸湿了我的指尖。睁开眼，一片清浅透亮的水潭呈现在眼前。

mysqlQuestions1

发表于 2021-05-23 更新于 2021-05-25

初始化正常，但是无法启动。手动启动mysql显示“错误2系统找不到指定的文件”。尝试mysql -u root -p时，出现cannot连接到“3306”（10060）。
解决方法：
打开regedit注册表。HKEY_LOCAL_MACHINE->SYSTEM->CurrentControlSet->Service->MySQL
找到ImagePath打开。修改“数值数据”。内容填写为mysql.exe的路径（在bin中）。保存后可以启动。

转青山

发表于 2021-03-15 更新于 2022-09-30 分类于 《转青山》

七月半，鬼门开。
长街小巷纵横的街道上，早早地摆了那烧袱纸的“聚宝盆”，用来给逝去的家人送些用度。家家户户还备了活计物什，只待到时辰将那锡箔折成锭，沿路焚化，好教自己“结鬼缘”、求平安。
入夜，月上了枝头，城中一派宁静。我拂开随风飘到的衣襟上的冥币，捡了小路往郊边昭王坟去。夜间雾浓蔽目，但此处我熟得很，闭着眼都能晓得我在何处。走出城门，复行数十步才到了一处柏树浓密的林子，放眼望去，左右尽是无名坟包，最大的那个上硬立了个墓碑。
这就是庭山脚下的昭王坟了。
